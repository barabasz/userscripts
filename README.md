# UserScripts

A collection of browser userscripts for use with [Tampermonkey](https://www.tampermonkey.net/), [Violentmonkey](https://violentmonkey.github.io/), or similar managers.

## Installation

1. Install a userscript manager (e.g., Tampermonkey)
2. Click the **Install** link next to the script
3. Confirm installation in the popup dialog

## Scripts

| Script | Description | Install |
|--------|-------------|---------|
| [wiki-remove-references.user.js](wiki-remove-references.user.js) | Removes citation markers from Wikipedia articles | [Install](https://github.com/barabasz/userscripts/raw/main/wiki-remove-references.user.js) |
| [wiki-ui-fixes.user.js](wiki-ui-fixes.user.js) | Removes various UI elements from Wikipedia pages (phonos, mboxes, mw-data-after-content, footer) | [Install](https://github.com/barabasz/userscripts/raw/main/wiki-ui-fixes.user.js) |
| [wiki-immersive-links.user.js](wiki-immersive-links.user.js) | Makes Wikipedia links more immersive | [Install](https://github.com/barabasz/userscripts/raw/main/wiki-immersive-links.user.js) |
